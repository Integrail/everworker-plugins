# Everworker plugins for Claude Code

Public marketplace for [Everworker](https://everworker.ai) Claude Code plugins.

## Install

```text
/plugin marketplace add Integrail/everworker-plugins
/plugin install ai-builder@everworker
```

See each plugin's own `README.md` for usage.

## Plugins

- [`ai-builder`](./plugins/ai-builder) — Plan, develop, and test Everworker AI Builder artifacts (workflows, Universal Workers, code nodes, memories, collections, webhooks, schedules) from Claude Code.

## Releases

This repo is a one-way mirror of plugin sources from the canonical
[`Integrail/everworker`](https://github.com/Integrail/everworker) repository.
Published via `scripts/release-plugin.sh` in that repo.
